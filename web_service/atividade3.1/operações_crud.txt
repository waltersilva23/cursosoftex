Operações com o CRUD


Criação (Create): 
Rota: POST /books Corpo da requisição:


{
    "name": "The Lord of the Rings",
    "author": "J.R.R. Tolkien",
    "isbn": "9780544003415"
}


Resposta: 201 Created Descrição: essa rota é usada para adicionar um novo livro ao estoque. O nome, autor e ISBN do livro são enviados no corpo da requisição e um ID único é gerado pelo servidor e retornado na resposta.


Leitura (Read): 
Rota: GET /books/:id Resposta: 200 OK Exemplo de resposta:


{
    "id": 1,
    "name": "The Lord of the Rings",
    "author": "J.R.R. Tolkien",
    "isbn": "9780544003415"
}

Descrição: essa rota é usada para obter informações sobre um livro específico, usando o ID do livro na rota. A resposta inclui todas as informações sobre o livro.




Atualização (Update):
 Rota: PUT /books/:id Corpo da requisição:

{
    "name": "The Lord of the Rings - Updated",
    "author": "J.R.R. Tolkien",
    "isbn": "9780544003415"
}


Resposta: 204 No Content Descrição: essa rota é usada para atualizar as informações de um livro específico, usando o ID do livro na rota. O corpo da requisição inclui as informações atualizadas do livro.


Exclusão (Delete):
 Rota: DELETE /books/:id Resposta: 204 No Content Descrição: essa rota é usada para excluir um livro específico do estoque, usando o ID do livro na rota. A resposta não inclui nenhum conteúdo, pois o livro foi excluído.
